## Free Math Practice


Freemathpractice is an unlocked github pages site that lets the user play the pokemon game of their choice, but our github pages comes preloaded with Pokemon Fire Red.


## Custom builds


To load your own gba files, clone this repo, remove the fire red.gba file and place your own gba file instead, github pages should automatically detect the change in the codebase, restart the page, and your new game will be deployed online for free, unlocked. 





## Github Pages


In your cloned repository inside of github
Settings > Pages (left side menu) 

You can use my settings if you want, otherwise set your own, wait for github pages to spin up, and play.
